# **App Name**: College Compass

## Core Features:

- AI Mentor Chat: AI-powered chat interface for students to ask questions about higher education. The LLM will use tools to browse various university and scholarship databases.
- Resource Library: Curated list of resources, including articles, videos, and links to external websites, related to college applications, financial aid, and academic success.
- Personalized Recommendations: Based on student profiles and chat history, provide tailored recommendations for colleges, scholarships, and resources.
- User Authentication: Secure user accounts for students to save their progress and preferences.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke trust and guidance.
- Background color: Very light blue (#F0F8FF) for a clean and calming interface.
- Accent color: Soft green (#98FB98) to highlight important resources and calls to action. This creates a good contrast against the light blue, while remaining in an analogous hue range.
- Font pairing: 'Inter' (sans-serif) for body text and 'PT Sans' (sans-serif) for headlines to combine modernity with readability.
- Use clean, modern icons related to education and career planning.
- Clean and intuitive layout with clear sections for the AI mentor, resource library, and personalized recommendations.
- Subtle animations for a smooth user experience, such as loading indicators and transition effects.
